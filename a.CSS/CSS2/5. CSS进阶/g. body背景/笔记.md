# body背景
**画布 canvas**
<canvas>元素

画布 一块区域  不仅仅是浏览器窗口

特点:

1. 最小宽度为视口宽度
2. 最小高度为视口高度

**HTML元素的背景**

覆盖画布  别的元素都是边框盒

**BODY元素的背景**

如果HTML元素有背景,body元素正常和其他元素一致(背景覆盖边框盒)
如果HTML元素没有背景,body元素的背景覆盖画布

**关于画布背景图 body和html的背景图**

1. 背景图的宽度百分比,相对于视口
2. 背景图的高度百分比,相对于网页高度(html元素的高度)
3. 背景图的横向位置百分比、预设值、相对于视口
3. 背景图的纵向位置百分比、预设值、相对于网页高度
